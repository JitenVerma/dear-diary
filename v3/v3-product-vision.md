🌙 Dear Diary v3 — Core Concept
Product Shift

v1 → record
v2 → understand
v3 → process + reset + prepare

🧠 New Product Identity

Dear Diary becomes a daily ritual space.

Not:

a journaling tool
a planner
a habit app

But:
👉 a place to arrive, reflect, release, and gently prepare for tomorrow

🧭 V3 CORE SYSTEM

Everything in v3 should revolve around ONE experience:

✨ “Daily Ritual”

A structured but flexible flow:

Arrive
Choose Mode
Write / Reflect
Prepare Tomorrow (optional)
Close the Day
🧩 1. Entry Modes (CORE FEATURE)

This is the foundation of v3.

Instead of:

“New Entry”

You have:

“How do you want to use this space today?”

🎭 Entry Modes Overview
Mode	Purpose	When user picks it
Reflect	Understand the day	calm / thoughtful
Unload	Release emotions	overwhelmed
Plan	Think about tomorrow	future-focused
Rest	Minimal / low effort	tired / low energy
Free Write	No structure	default / flexible
🧠 Mode Selection UX
Entry point

User clicks:
👉 “New Entry”

Instead of opening editor → show Mode Selection Screen

Mode Selection Screen

Soft UI:

How are you arriving today?

[ Reflect ]   “Make sense of your day”
[ Unload ]    “Let things out”
[ Plan ]      “Prepare for tomorrow”
[ Rest ]      “Keep it light”
[ Free Write ] “Just write freely”
Large cards
soft descriptions
no pressure
user can skip → Free Write

🌙 3. Closing the Day (SIGNATURE FEATURE)

This is your most emotionally powerful feature

🧠 Concept

After writing, user is guided to:

👉 “Close the day”

✨ Closing Ritual Flow

After entry:

Would you like to close the day?
Step 1 — Day Reflection
Today, I would describe this day as:

[ peaceful ]
[ heavy ]
[ unfinished ]
[ meaningful ]
[ tiring ]
[ hopeful ]

(Custom or selectable)

Step 2 — Carry / Release
What am I carrying into tomorrow?

What am I leaving behind tonight?
Step 3 — Final Note
A quiet note to myself...
Step 4 — Close

User clicks:
👉 “Close the Day”

🎬 Visual Experience

This is IMPORTANT:

background softens
timeline subtly animates
entry “settles” into timeline
gentle fade / glow
maybe a slight dimming effect

👉 Should feel like:
putting the day to rest
